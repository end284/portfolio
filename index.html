My Projects

Smart Bin/h3>

An innovative smart trash bin that opens and closes its lid automatically when it detects an object or hand nearby. This project uses an ultrasonic sensor to measure distance, allowing users to dispose of waste without touching the lid, making it more hygienic and modern.

Self Driving car

A mini autonomous car designed for the World Robot Olympiad (WRO) 2025 in the Future Engineers category. The car is equipped with sensors, cameras, and programming algorithms that enable it to navigate independently, recognize paths, and avoid obstacles. This project focuses on advancing skills in robotics engineering and intelligent transportation systems.

Smart Dry-G

A prototype simulation of an intelligent grain drying system that utilizes temperature and humidity sensors to monitor the drying process. The system is connected to Blynk IoT, allowing real-time data monitoring through a mobile application. This project demonstrates how technology can support agriculture by providing more efficient and controlled grain drying compared to traditional methods.

Smart Window

A smart window prototype that can automatically open or close based on environmental conditions. The system uses sensors such as light and rain detectors to improve energy efficiency and safety inside the house. It can also be controlled manually via a mobile app for convenience. This project showcases the integration of IoT technology into daily life for smarter home automation.
